* **課程概述與目標：**<br>
本課程著重在教與基礎機器學習和深度學習觀念，讓學生可以從機器學習/深度學習的根本來認識AI在做什麼，如何達到Data driven的學習程序。<br>
上課內容會依據上課同學的反應和實際狀況，進行滾動式調整。<br>

* **課程專題：**<br>
專題提案，每組同學都要提出想法(一頁至三頁的投影片資料)，會由三位老師來評估專案的實作可行性。<br>
會進行兩次期末專題發表。<br>
由三位老師評分。
